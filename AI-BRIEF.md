# 🤖 AI协作工作区

## 给Codeium的工作须知
- 用户是零基础学习者，请写简单易懂的代码
- 代码注释用中文，变量名用英文
- 每段代码都要解释它干了什么
- 遇到复杂逻辑，分成小步骤写

## 给巴特鹿的反馈说明
- 我会在 FEEDBACK.md 里写反馈和指导
- 你写完代码后，等1-2分钟我就能看到
- 有问题随时在聊天里问我

## 项目结构
```
ai-workspace/
├── AI-BRIEF.md      ← 你现在看的这个（协作须知）
├── auto-sync.bat     ← 双击启动自动同步
├── FEEDBACK.md       ← 巴特鹿的反馈（自动更新）
├── src/              ← 你的代码放这里
└── notes/            ← 学习笔记放这里
```
